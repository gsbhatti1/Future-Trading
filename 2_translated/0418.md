> Strategy Description

Set an amount of capital for investment, e.g., 10,000 yuan. Set a ratio of capital to the value of Bitcoin, e.g., 50:50. Maintain this ratio according to the current price.

For example, if the initial capital is 10,000 yuan and the capital-to-Bitcoin ratio is set at 30:70, in the first transaction, 7,000 yuan worth of Bitcoin (assuming a coin price of 1,000 yuan, buying 7 coins) will be purchased. The remaining capital would be 3,000 yuan and the ratio should remain at 30:70.

In the next trading cycle, if the coin price rises to 1,500 yuan, the total value becomes 1,500 * 7 + 3,000 = 13,500. To maintain the capital-to-value ratio of 30:70, the required capital should be 13,500 * 0.3 = 4,050 yuan, so 4,050 - 3,000 = 1,050 yuan worth of Bitcoin needs to be sold, which equals 1,050 / 1,500 = 0.7 coins. Thus, the number of bitcoins left would be 7 - 0.7 = 6.3 coins with a value of 6.3 * 1,500 = 9,450 yuan. The capital-to-bitcoin value ratio remains at 4,050:9,450 = 30:70.

Similarly, when the price falls, buy to maintain the same ratio.

> Strategy Arguments


|Argument|Default|Description|
|----|----|----|
|UseAccount|true|Use all funds in the account|
|InitMoney|10000|Initial capital amount.|
|InitStock|5|Initial number of coins.|
|StockRatio|0.418|Percentage of coin value to total value ratio|
|InvestInterval|true|Investment interval.|
|ErrorInterval|300|Error retry interval.|
|SlidePrice|0.01|Slippage|

function CaculateHoldings() {
    CancelPendingOrders();

    if (lastOrderID) {
        var order = exchange.GetOrder(lastOrderID);

        if (order.Status === ORDER_STATE_CLOSED) {
            if (order.Type === ORDER_TYPE_BUY) { //Buy
                stockHold += order.DealAmount;
                moneyHold -= order.DealAmount * order.Price;
                Log("Order Buy. stockHold +=:", order.DealAmount, ',Money -=:', order.DealAmount * order.Price);

    }

    var currentPrice = GetCurrentPrice();
    prices.push(currentPrice);
    //Log('History prices:',prices);

    var stockValue = 0;
    var totalValue = 0;
    var ratio = 0;

    stockValue = stockHold * currentPrice;
    totalValue = stockValue + moneyHold;

    ratio = stockValue / totalValue;
    Log('Current Price:', currentPrice);
    Log('stockHold:', stockHold);
    Log('moneyHold:', moneyHold);
    Log('stockValue:', stockValue);
    Log('Current Ratio:', ratio);

    if (ratio > StockRatio) { //Sell
        Log('Current ratio > StockRatio');
        var sellAmt = (stockValue - totalValue * StockRatio) / (currentPrice - SlidePrice);
        lastOrderID = exchange.Sell(currentPrice - SlidePrice, sellAmt);
    } else { //Buy
        Log('Current ratio < StockRatio');
        var buyAmt = (totalValue * StockRatio - stockValue) / (currentPrice + SlidePrice);
        lastOrderID = exchange.Buy(currentPrice + SlidePrice, buyAmt);
    }
    Log('Order ID:', lastOrderID);
    CaculateHoldings();

}
